# 💎 داشبورد مشروع الجوهرة

## التشغيل (خطوتان فقط)

### 1. تثبيت المكتبات (مرة واحدة)
```
pip install -r requirements.txt
```

### 2. تشغيل الداشبورد
```
streamlit run dashboard.py
```
سيفتح المتصفح على: http://localhost:8501

---

## المحتوى

| التبويب | المحتوى |
|---|---|
| 📊 ملخص | KPIs + S-Curve + توزيع الأقطار + تنبيهات |
| 🏗️ المناطق | تنفيذ كل منطقة + تفاصيل الخطوط |
| 📋 التصاريح | قائمة التصاريح + بحث + تحميل CSV |
| 🗺️ الخريطة | Google Maps + KMZ تفاعلي |
| 📈 تحليلات | القيمة المنفذة + نسب الإنجاز |
